# Hoobaan Construction Company Website

Static website for Hoobaan Construction Company, built with HTML, Tailwind CDN, JavaScript, images, and video assets.

## Pages

- `index.html` redirects to the home page.
- `index.html` is the landing page.
- `about.html` introduces the company and team.
- `services.html` lists construction services.
- `Projects.html` showcases completed projects, 3D visuals, interiors, and video.
- `AdnaDhiso.html` contains the client project request form.

## Deployment

This project can be deployed as a static site on GitHub Pages or Netlify.

For Netlify:

- Build command: leave empty.
- Publish directory: project root.

## Notes

- Large video assets should stay under GitHub's 100 MB file limit.
- The `Adna Dhiso` form opens a prepared WhatsApp message with the client's form details.
- Local editor and macOS files are ignored through `.gitignore`.
